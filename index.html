Truth Table Generator

Given a set of variables and a logical expression, the site should generate its corresponding truth table.

Input the variables you would like to use in your expressions. Variables must be single uppercase letters. Click "Continue" when you are done setting all the variables.

Enter the expression you would like to add to the truth table.
To help with typing in operators, n → ∼ (not), a → • (and), o → ∨ (or), i → ⊃ (implies), and e → ≡ (equal).
In addition, t → 𝑡 (true) and f → 𝑓 (false).
